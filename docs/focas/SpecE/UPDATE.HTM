Update History of FOCAS1/2 Library

Edition 5.3 (December 25, 2019)

  • On 64-bit Windows 10 operating system, When the FOCAS application using the FOCAS function for NC guide accesses the CNC guide, application error may occur. This problem has been corrected.
  • When the file information acquisition function cnc_rddsfile of the data server is executed in the Fast Ethernet connection, the number of acquired files may not match the number of acquired files. This problem has been corrected.
  • In FOCAS2 / Ethernet, a library handle allocation function cnc_allclibhndls corresponding to the security function by the access code has been added.
  • FOCAS application can be operated by multi-user on PC.
  • In the servo waveform acquisition function, the exclusive control of the sampling function is supported.

Edition 5.2 (April 25, 2019)

  • A license agreement to use FOCAS library is revised.

Edition 5.1 (November 21, 2018)

  • A license agreement to use FOCAS library is revised.
  • Fwlib30i64.dll : When executing PMC functions such as pmc_rdpmcrng function of 64-bit FOCAS2/HSSB in Windows10, the program exception might occur. This problem is corrected.
  • The following libraries are deleted.
    • FOCAS2 library for Linux
    • FOCAS2 library for Android
    • FOCAS2 library for iOS

Edition 4.16 (June 05, 2018)

  • A license agreement to use FOCAS library is revised.
  • Fwlibe1.dll : When reading spindle data by cnc_rdaxisdata function of FOCAS2/Ethernet, the incorrect data might be read in data of the second or more axes in a path and the second or more types. This problem is corrected.
